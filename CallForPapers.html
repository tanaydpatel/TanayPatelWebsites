ICPC 2020 welcomes original and innovative paper submissions from the academia, industry, and government, describing theoretical advances, system designs, implementations, and experimentations, including but not limited to the following topics:

The broad area of conference includes, but not limited to,

Track 1 : Pervasive Computing and Cloud Computing

  • Middleware systems and services.
  • Data engineering for pervasive computing.
  • Cloud computing.
  • Pervasive computing : healthcare; cyber-physical computing; innovative computing applications.
  • Storage Area and Networking.
  • Virtualization Infrastructure and Platforms.

Track 2 : Data Analytics and Machine Learning

  • Active learning
  • Computational learning theory
  • Distance measurement learning
  • Deep learning
  • Machine learning
  • Artificial Intelligence
  • Data mining
  • Big data analytic

Track 3 : Signal, Image and Video Processing

  • Image coding and compression
  • Pattern recognition and Analysis
  • Analog and Mixed signal processing
  • Audio/speech processing
  • Digital and mobile signal processing
  • Video signal processing
  • Communication signal processing
  • Image and video retrievals

Track 4 : IoT and Communication

  • IoT standardization : Semantic service, security, privacy and trust
  • Next generation networks
  • IoT Led Digital Transformation
  • Cloud Enabled Remote Management Solution for IoT Edge
  • Field Area Networks for Utility Automation
